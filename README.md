# Agent Prompts

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A personal collection of reusable AI prompts for coding agents (pi, Claude Code,
Codex, etc.) and chat applications (ChatGPT, Claude.ai, Perplexity, etc.).

Each prompt lives in its own file or directory, organised by tool or category,
so you can copy only what you need.

## Usage

Browse the prompts and copy the content into your agent or chat interface.
Each prompt file includes a header comment describing its purpose and intended
use.
